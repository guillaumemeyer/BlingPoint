    • BlingPoint

    BlingPoint is a JS runtime & framework for easily building instrumented, modular and robust SharePoint solutions that relies on the best design principles.

    Project maintained by Guillaume Meyer Proudly sponsored by ERYEM

    FEATURES

    BlingPoint provides an easy to use, lightweight and efficient library of tools and features to ease development of powerful, modular & robust SharePoint solutions :

    • Embedded logging & diagnostic system
    • Simple profiling tool
    • PlugIn Management System
    • Schema provisioning (Sites, lists, firlds, content types...)
    • Application parameters Management
    • Scripts & CSS dynamic loading
    • Script-oriented syntax
    • Simple URL tools
    • More to come... Stay Tuned !


    GET STARTED

    This procedure is for testing purposes only ! For production use, deploy BlingPoint to your own servers.

    1. Add the BlingPoint bookmarklet
    Start by adding this link to your bookmark bar: Launch BlingPoint

    2. Launch BlingPoint
    • Navigate to your own site as an administrator.
    • Click on your newly created bookmark "Launch BlingPoint"
    • If everything is OK, you will soon see the BlingPoint log viewer window appear


    3. Your first exercise, create a new site
    Follow this procedure to create a new blank site with just one single command.


    DEPLOYMENT

    As a single minified javascript file, BlingPoint is easy to deploy :

    • By manually registering a script tag in your site MasterPage
    • Using this bookmarklet to inject it dynamically into any SharePoint page (For testing purposes only !): Launch BlingPoint
